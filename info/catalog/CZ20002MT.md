## CZ20002MT Токарный станок с подачами и увеличенным расстоянием от станины с мотороным блоком с шаговым двигателем

Three-axis CNC lathe CZ20002MT

Features:
1. Numerical control system: mach3 (or Zhouyu industrial system)
2. Main parts of machine tool: such as spindle case, middle block, motor Vane, gear, small slider (cross sliding), long slider, tail seat, connecting block, two-section knife seat, long machine seat and so on all adopt metal materials.
Technical indicators:
1. The main axis is the rotating axis (A Axis): A = 360 °/200/8 = 0.225 °
2. X, Z, A Axis stepping motor: 1.5Nm, current 2A.
3. Connect the computer port: parallel port
4. Control box Power supply: control box Power supply: input AC110V-240V
