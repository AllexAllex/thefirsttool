## Z8000 8 базовых пластиковых станков

![Z8000](Z8000.webp)

8 In 1 basic mini machine kit Z8000

The eight-in-one set can assemble 8 kinds of machine tools with different functions (line sawing machine, wood lathe, metal lathe, milling machine, drilling machine, Sand Mill, hand-held machine tool, grading drilling machine ), A machine tool that can only use one of the functions at a time.
 1. Motor speed: 20000 revolutions per minute. 
2. Input voltage/current/power: 12VDC/2A/24 W, switching power's switching voltage is 110 V-240 V. It is a good tool for family and school children to make aerial models and other woodworking models
