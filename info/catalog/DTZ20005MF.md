## DTZ20005MF Станок для изготовления "бусин"

![DTZ20005MF](DTZ20005MF.webp)

Big power LCD digital Buddha beads machine DTZ20005MF

Features: 
The machine tool adopts motor with more powerful horsepower, and the motor box and the spindle box are in joint structure, which can process Buddha beads. Main parts of the machine tool: such as the spindle case, the long engine base, the short engine base, the size slider, the motor fan blade, the connecting block, the Tiger tongs, the gears and so on all adopt metal materials, using the digital liquid crystal display instead of the scale reading, it avoids the disadvantages of difficulty in reading scale, large parallax and easy to forget numerical value. Clear hand wheel type liquid crystal reading display (the liquid crystal reading is in the hand wheel position, not the external LCD reading) is greatly convenient for users to read processing data while processing, the ZERO position and machining starting point position can be set at any slider position through the ZERO key. It has the one-key conversion function of Metric/British, which is convenient for users to have different habits. It has the memory function. When it stops using for one minute, it will automatically cut off the power, and when the user presses the key again, it will automatically display the processing value when the power is cut off. The display accuracy can reach 0.01mm, and LCD display is powered by a 1.5V battery. 
Technical parameters: 
1. Rotate speed of motor: 12000 revolutions per minute. 
2. Input voltage/current/power: 12VDC/5A/60 W. 
3. Processing materials: wood, plastic, organic glass, soft metal (gold, silver, copper, aluminum) 10. The biggest tool that can process the metal Buddha beads with a diameter of 20mm is the Buddha beads knife. The side and front of the Buddha beads knife are blade, which is dangerous to some extent, therefore, it must be used under the guidance of professionals.
