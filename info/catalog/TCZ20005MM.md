## TCZ20005MM горизонтальный фрезерный станок

![TCZ20005MM](TCZ20005MM.webp)

Технические параметры:

- Система: Mach 3 (дополнительная промышленная система Zhouyu)
- Скорость двигателя: 12000 об./мин.
- Входное напряжение/ток/мощность: 12VDC/5A/60 Вт
- Ось Y, Z ползунок Путешествия 50 мм, ось X 145 мм
- Цанги: 1-6 мм
- Максимальный диаметр заготовки, зажатый машинными тисками: 50 мм
- X. Y. Zstepping Мотор: 1.5Nm, ток 2A, 1.8deg
- Точность: 0,07 мм
- Рабочий материал: Лигнин пластик, мягкий металл (золото, серебро, медь, алюминий и т. д.)
- Компьютерный порт: параллельный интерфейс
- Мощность контроллера и адаптера: Входной AC110-240V, выход DC12V, 5A

Horizontal CNC milling machine TCZ20005MM

Features: The machine tool adopts motor with more powerful horsepower, and the power of the main shaft motor can reach 144 W. The Motor box and the main shaft box are of joint structure. 
1. Numerical control system: mach3 (or Zhouyu industrial system) 2. Main parts of machine tool: such as spindle case, middle block, motor fan blade, gear, small slider (cross sliding), long sliding block, connecting block, long machine base, short machine base, milling machine with Tiger pliers and so on all adopt metal materials. Technical indicators:
 1. Motor speed: 12000 revolutions per minute
2. Input voltage/current/power/: 12VDC/5A/70W 
3. X, Y, Z stepping motor: 1.5Nm, current 2A, 1.8 °
4. Connect computer port: parallel port 
5. Control box Power supply: input AC110V-240V
