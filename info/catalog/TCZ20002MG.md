## TCZ20002MG Токарный станок с подачами и увеличенным расстоянием от станины

![TCZ20002MG](TCZ20002MG.webp)

Metal CNC lathe B Elevation TCZ20002MG

Features: 
The machine tool adopts motor with more powerful horsepower, and the power of the main shaft motor can reach 144 W. The Motor box and the main shaft box are of joint structure.
1. Numerical control system: mach3 (or Zhouyu industrial system) 2. Main parts of machine tool: such as spindle case, middle block, motor fan blade, gear, small slider (cross sliding), long slider, tail seat, two-section knife seat, connecting block, long machine seat and so on all adopt metal materials.
Technical indicators:
1. Motor speed: 12000 revolutions per minute
2. Main shaft motor input voltage/current/power/: 12VDC/5A/70W 3. Main shaft motor power adapter input voltage AC110V-240V, output voltage DC12V current 5A.
4. Connect to the computer port: parallel port
