## Z20005M Фрезерный станок

![Z20005M](Z20005M.webp)

Mini Metal Milling Machine Z20005M

 Feature:
1. Milling cutter, with two edges, is dangerous, so people should be guided by experts when using it。
2. Main parts: headstock, long machine bed, short machine bed, big slider, small slider, motor blade, connection piece, gear etc. All of the parts are made of metal.     
Technical parameters:
1. Motor speed :20,000rpm/min(optional choice 12,000rpm/min)
2. Input voltage/current/power:12VDC/2A/24W
3. Axis Y,Z travel :30mm   Axis X travel:160mm
4. Collets: 1-6mm
5. Maximum working material size clamped by machine vice:50mm
6. With 0.02mm scale line ,the hand wheel increases the precision when processing.
7. Working material: wood, plastic, soft metal (gold, silver, copper, aluminum etc.), acrylic, plastic etc
