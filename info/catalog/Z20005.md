## Z20005 Фрезерный станок

![Z20005](Z20005.webp)

Milling Machine Z20005

 Feature:
1.The machine can be operated in X,Y,Z three dimensional space, can be assembled into vertical milling machine, horizontal milling machine and hand-held milling machine.  
2. Milling cutter, with two edges, is dangerous, so people should be guided by teachers when using it. 3.Machine color is made up of red and black.
Technical parameters:
1.Motor speed :20,000rpm/min ,motor and gear are made of metal. 2.Input voltage/current/power:12VDC/2A/24W
3.Slider travel:30mm，50mm
4.Collets：1-6mm
5、Working material size clamped by machine vice:30mm  6.Working material: wood, engineering plastic, soft metal(aluminum, copper etc.)
7. The transformer has over-current protection, over-voltage protection, over heating protection
